## 📝 문제 요약

N×N 격자에 우주 괴물(2)이 한 마리 있고, 괴물은 상·하·좌·우 네 방향으로
장애물(1)이나 격자 경계를 만나기 전까지의 모든 칸을 공격한다.
공격받지 않는 안전한 빈 칸(0)의 개수를 구한다.

## 💡 접근 방법

1. `enumerate`로 격자를 순회하며 값이 `2`인 괴물의 위치를 찾는다.
2. 괴물 위치에서 상·하·좌·우 각 방향으로 `while` 루프를 돌며 한 칸씩 전진한다.
   - 빈 칸(0)이면 공격 범위로 표시(`1`로 마킹)하고 계속 전진한다.
   - 0이 아닌 값(장애물 등)을 만나면 그 방향 탐색을 멈춘다.
3. 마킹이 끝난 뒤 격자에 남아 있는 `0`의 개수를 세어 출력한다.

## ⏱️ 복잡도

- **시간**: O(N²) — 괴물 탐색과 최종 카운트가 격자 전체 순회 (방향 마킹은 O(N))
- **공간**: O(N²) — 격자 저장
